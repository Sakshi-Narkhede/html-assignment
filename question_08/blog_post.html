My Journey with HTML

By Sakshi Narkhede, Posted on May 12, 2025

Introduction:

I was facing a problem in everyday life, & couldn't really find any proper solution for it. That's when I thought, maybe I can turn this into a project. It might help others who are facing the same issue.

While looking into how to get started, I found out that I'd need to create a website, and for that, learning HTML was important.

So, I began learning HTML to better organize my idea through a website.

My Learning So Far

I've learned the basics of HTML including:

  • Headings (from <h1> to <h6>)
  • Paragraphs
  • Forms
  • Tables
  • Lists
  • Semantic tags like <header>, <aside>, <nav>, and more

My Future Plans

I'm planning to learn JavaScript along with CSS. Starting JavaScript early is important since it's a bit tricky but super useful — not just for frontend work, but for backend development too.
